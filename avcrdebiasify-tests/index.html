K čemu to je. Toto je technický test propojení s nástrojem Debiasify, který v textu hledá rétorické (jak je něco napsáno) a faktické (co a jak je doloženo) znaky ovlivňování. Příspěvky jsme automaticky načetli z oficiálního účtu Akademie věd ČR na síti Bluesky (akademievedcr.bsky.social) a poslali k rozboru. Tuto síť používáme proto, že je její obsah přímo strojově přístupný přes veřejné API — na rozdíl od Facebooku, který automatické čtení cizích stránek a komentářů zvenčí neumožňuje. U příspěvků jsme analyzovali i reakce čtenářů pod nimi — pouze ty delší než 100 znaků, které nástroj dokáže vyhodnotit (kratší komentáře Debiasify analyzovat neumí). Reakce jsou vnořené pod daným příspěvkem. Nejde o hodnocení Akademie věd — jde o ukázku, jak se nástroj chová na reálných, převážně popularizačních a PR textech. Vysoké „PR“ nálezy jsou u oficiální komunikace běžné a neznamenají dezinformaci.
Jak číst procenta — a co je dobré a co špatné.

U všech tří metrik platí stejný směr: 0 % = nejlepší (věcný, neutrální text bez manipulace), 100 % = nejproblematičtější (silné ovlivňování). Čím vyšší číslo, tím hůř.

  • Rétorika (0–100 %) — jak je text napsán: emoce, naléhavost, útoky na osobu, efektní rámování. 0 % = žádný rétorický trik (dobré), 100 % = jednoznačný silný rétorický nátlak (špatné). Číslo je míra jistoty u nejsilnějšího nálezu, ne podíl „zaujatého“ textu.
  • Fakta (0–100 %) — co a jak je doloženo: chybějící informace, vágní nebo chybějící zdroje, nepodložená tvrzení, skrytá reklama. 0 % = nic podstatného nechybí (dobré), 100 % = vážné faktické vady či zamlčování (špatné).
  • Obě hodnoty jsou stupňované: 0 % nic · 25 % slabý náznak · 50 % střední · 75 % silný · 100 % jednoznačný nález.
  • Míra ovlivňování (0–100 %) — souhrnně, nakolik použité techniky snižují informační hodnotu textu. 0 % = čistě věcné a informativní (nejlepší), 100 % = převažuje manipulace nad obsahem (nejhorší). Slovní stupně: zanedbatelná → nízká → střední → vyšší → vysoká.
