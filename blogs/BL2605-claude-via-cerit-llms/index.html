Claude Code on Czech academic AI

Route Claude Code's agentic tool-calls to the free CERIT LLM gateway via a local rewrite proxy. Zero API cost for MetaCentrum / MUNI members.

TL;DR
  • Free LLM API at llm.ai.e-infra.cz — MetaCentrum or MUNI account required
  • A local proxy handles model routing, tool compatibility, and idle-stop prevention
  • Best model: GLM-5.2 — 494 s · 7/7 tasks · 0 idle stops · 0 errors

What CERIT provides

CERIT-SC / e-INFRA CZ operates llm.ai.e-infra.cz — an Anthropic-compatible LLM API endpoint free of charge for Czech academic users holding a MetaCentrum or MUNI account. All inference runs on national infrastructure; no data leaves Czech academic networks.

Why a proxy is needed

  • Tool-type compatibility — Claude Code sends Anthropic-proprietary tool types the CERIT endpoint rejects (HTTP 400). The proxy strips and converts them before forwarding.
  • GLM-5.2 thinking mode — GLM-5.2 ships with thinking enabled by default, which causes ~900 s timeouts on agentic sessions. The proxy disables it per-request.
  • Model name / TUI — Claude Code's progress spinner requires ANTHROPIC_MODEL to start with claude-. The proxy handles the spoofed name while routing to the real CERIT model.

How the proxy works

The proxy intercepts every request between Claude Code and the CERIT endpoint and applies six interventions:

  1. Tool sanitizer — strips unsupported Anthropic tool types, patches missing input_schema fields
  2. Continuation injection — appends an agentic system-prompt block so the model calls tools instead of narrating; model signals task completion with TASK_COMPLETE
  3. Turn + repetition guards — prevents runaway sessions (synthesis nudge at turn 12, hard stop at turn 20, loop-break on repeated tool calls)
  4. GLM thinking disable — injects chat_template_kwargs: {"enable_thinking": false} for GLM-5.2 targets
  5. Context-overflow fallback — on HTTP 400: glm-5.2 → kimi → qwen3.5-122b
  6. HTTP 429 retry — 3× retries with 5 s backoff; thread-safe request counter

The continuation block injected into every system prompt:

CERIT_CONTINUATION snippet
AGENTIC EXECUTION — NON-NEGOTIABLE
When tools are available: call a tool immediately. Do not explain. Do not narrate.
end_turn is ONLY valid: (a) task 100% done — write your FULL final answer now,
  OR (b) you need information only the human can provide.
All other turns: CALL A TOOL. No exceptions.

Setup — step by step

bash
# 1. Get a MetaCentrum or MUNI account → token at llm.ai.e-infra.cz
mkdir -p ~/.config/cerit
echo 'YOUR_TOKEN' > ~/.config/cerit/token && chmod 600 ~/.config/cerit/token

# 2. Clone and install
git clone https://github.com/mcer33/claude-code-cerit-llm ~/dev/claude-code-cerit-llm
cd ~/dev/claude-code-cerit-llm && bash install.sh

# 3. Reload and verify
source ~/.bashrc && claude-cerit-ping

# 4. Launch
claude-cerit-rich
